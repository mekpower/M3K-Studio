Chat

1. How Install

  1. When you click on the link, this popup will appear;
  2. Click in “use” to apply to your current theme;

  3. Go to the chat box and done!

  4. Get the launch link and put in your streaming software.

2. Time show message

  1. To change the time of the message, go to the option "Hide Message After" and change the slide. To always show the message, check "Always Show Messages".

3. To Customize

  1. When scrolling down, you will see a bar,with a label written "Custom Field", click on it and make the changes you want.

4. How use multiple styles at same time?

  1. Add the other chat style in a new theme. After that click on the link icon and copy the chat. Put it in the scene that that style belongs.

Goals

Start a goal

To start a new goal. That work for any type of goal.
All Widgets > type goal> Manage Goal , click in button End Goal. Setup the details of the goal.

Change to custom style

The goal is come using the simple style. To get the custom style, click in My Goal.

To change colors and font size, go to:
All Widgets > type goal> Settings > Custom Fields

How use multiple styles at same time?

Add the other widget style in a new theme. After that click on the link icon and copy the widget. Put it in the scene that that style belongs.
